23.4.2023
gitk
Нужен чтобы достать х-файл погребенный под релизами.
1) сделай копию позднейшего релиза
2) зайди в папку. сделай ПКМ-[git bash here]
3) запустится черная консоль. пиши gitk
4) запустится git-браузер. Сверху список всех коммитов/мерджей, и их id
5) скопируй нужные id [ctrl+ins]копировать [shift+ins]вставить
6) В туже черную консоль пиши 
git checkout ga4565   ....id

произойдет откат
7) скопируй х-файл в надежное место
8) скопируй содержимое папки из пункта(1) обратно средствами виндоус.
9) commit . Вуаля
